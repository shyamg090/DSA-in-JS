# 📘 What is an Algorithm?

This document contains my learning notes on **what an algorithm is**, its characteristics, and why learning algorithms is important for developers.

---

## 🧠 What is an Algorithm?

An **algorithm** is a set of well-defined instructions used to solve a particular problem.

### 🔹 How it works

An algorithm:

1. **Takes input**
2. **Processes the input through a series of steps**
3. **Produces an output**

### ✅ Example

* **Adding two numbers**

  * Input → Two numbers
  * Process → Perform addition
  * Output → Result of the addition

---

## ⚙️ Characteristics of an Algorithm

A good algorithm should have the following properties:

### 1️⃣ **Defined Inputs and Outputs**

* Clear understanding of what goes in and what comes out.
* No ambiguity in input or output.

### 2️⃣ **Clear and Unambiguous Steps**

* Instructions must be precise.
* Steps should be easy to understand and follow.

### 3️⃣ **Language Independent**

* Algorithms are not tied to any specific programming language.
* They can be implemented in JavaScript, Python, Java, or any other language.

---

## 🎯 Why Learn Algorithms?

### 💡 Problem Solving

* Developers constantly encounter problems that require solutions.

### ⚡ Efficiency

* Algorithms help you solve problems in efficient and optimized ways.

### ⚖️ Performance Trade-offs

* Multiple algorithms can solve the same problem.
* Each algorithm may differ in performance, speed, or resource usage.
